# 🌱 새싹 (SAESSAK)

> **새롭게 싹 지우고, 새싹을 틔우세요**
> 디지털과 지구에 숨 쉴 공간을 만드는 디지털 탄소 데이터 플랫폼

---

새싹은 귀찮은 디지털 데이터 정리를 AI가 대신 찾아주고, 절감된 용량을 **탄소 절감량**으로 환산해 **포인트(=새싹) 보상**으로 연결하는 **통합형 디지털 정리 플랫폼**입니다. 정리할수록 지구도 가벼워지고, 사용자도 혜택을 받습니다.

---

## 🧩 Problem — 이런 알림, 한 번쯤 받아보셨죠?

> 📩 *"Gmail 저장 공간 95% 사용 중 — 곧 새 메일을 받을 수 없게 됩니다"*
> 📱 *"iCloud 저장 공간이 가득 찼습니다 — 사진 및 동영상이 더 이상 백업되지 않습니다"*
> ☁️ *"저장 공간 부족 — 사진을 찍으려면 기존 파일을 삭제해야 합니다"*

**데이터는 계속 쌓이는데, 정리는 계속 미뤄집니다.**

### 📊 그래서 왜 안 지울까? — 자체 설문 (n=82, 20대 한정)

| 응답 | 비율 |
| --- | :---: |
| 정리 과정이 번거롭다 | **71.6%** |
| 실수로 중요한 파일을 삭제할까봐 걱정된다 | **46.6%** |
| 어떤 파일을 삭제해야 할지 모르겠다 | **43.2%** |

> **75%** 저장공간 부족 불편 경험 · **94%** 정리를 거의 안 하거나 급할 때만 함 · **60%** 별도 정리 수단 없이 방치

## 💡 Solution — 새싹은 정리를 더 쉽고, 지속 가능하게 바꿉니다

기존 서비스는 *메일 정리만*, 혹은 *탄소 측정만* 가능하고, 사용자가 직접 정리해야 하며 행동을 지속할 동기가 부족했습니다. 새싹은 네 가지를 한 번에 묶어 **선순환 구조**를 만듭니다.

| 단계 | 내용 |
| --- | --- |
| **AI 자동 분석·추천** | AI가 영역별 데이터를 분석하고 불필요한 항목을 추천 |
| **스와이프로 간편 정리** | 추천 항목을 좌(보관)/우(정리) 스와이프로 빠르게 처리 |
| **탄소 절감** | 정리한 데이터를 탄소 절감량(g CO₂)으로 환산해 시각화 |
| **리워드 제공** | 정리 → 포인트(=새싹) → 친환경 마켓 보상으로 지속 동기 부여 |

```
데이터 정리  →  디지털 탄소 절감  →  새싹(포인트) 적립  →  리워드 교환
(AI 추천 정리)   (절감량 시각화)      (절감량 비례)         (친환경 브랜드 혜택)
```

## ✨ Key Features

### 1️⃣ AI 디지털 탄소 측정

- **3개 영역 자동 스캔** — 이메일, Cloud 사진, Cloud 파일
- **AI 분석으로 탄소 환산** — 데이터 유형별 배출량 분석
- **월별 리포트 제공** — 디지털 탄소 추이를 한눈에

> 화면: `Connect` (계정 연결) · `Home` (오늘의 탄소) · `Analyze` (월별 분석)

### 2️⃣ 카드 스와이프 정리

- **AI 추천 카드 제공** — AI가 불필요하다고 판단한 데이터를 카드로 노출
- **스와이프로 간편 정리** — 좌(보관)/우(정리) 한 손 인터랙션
- **삭제 대기함(휴지통) 이동** — 친환경 시간대(새벽 02:00, 재생에너지 비중↑)에 자동 영구삭제 → 탄소 60% 절감

> 화면: `Recommend` (AI 추천) · `Result` (정리 결과·새싹 적립) · `Trash` (삭제 대기함)

### 3️⃣ 리워드 마켓

- **정리 → 포인트 적립** — 정리한 용량에 비례해 새싹 적립
- **다양한 보상 제공** — 커피 쿠폰, 쇼핑 할인, 환경단체 기부 등
- **나의 리워드 현황** — 누적 포인트와 혜택 내역을 한눈에

> 화면: `Green Market` (친환경 리워드 마켓) · `Exchange` (포인트 교환)

## 💰 Business Model — 공공 협력 기반 수익 모델

```
정부·기관  →  새싹 (플랫폼)  →  사용자
```

| 정부·기관 | 새싹 (플랫폼) | 사용자 |
| --- | --- | --- |
| 정부 보조금<br>*(청년그린창업·에코스타트업)* | 플랫폼 운영<br>*(ESG 데이터 수집·운영)* | 데이터 정리 참여<br>*(메일·사진·파일 정리)* |
| 기업 후원·스폰서십<br>*(브랜드 노출·친환경 이미지)* | 리워드 시스템 운영<br>*(포인트·친환경 마켓)* | 탄소 절감 실천<br>*(자동 측정으로 확인)* |
| | | 포인트 적립·사용<br>*(친환경 리워드 교환)* |

수익원

1. **브랜드 제휴 수수료** — 포인트 사용 시 제휴 비용 발생
2. **캠페인 협업 수익** — ESG 캠페인 운영 및 리워드 비용 지원
3. **Premium 구독(선택)** — 자동화·고급 기능 이용 시 구독 수익
